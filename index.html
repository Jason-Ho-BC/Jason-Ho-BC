3rd and 4th Mastery cores!

The long awaited Summer patch for MapleStory GMS has arrived. The main attraction is the completion of Mastery Cores with the 3rd and 4th Cores being added at the same time, but there'salso aton of game changes and improvements like the new Genesis Weapon Liberation changes, and the Skill Sequence system which lets you buff up with a single key!

Skill Sequence

With a single button you can active up to 15 skills at once! You can use the Skill Sequence by registering skills in your desired order. You can save multiple skill combinations and use them in different situations.
