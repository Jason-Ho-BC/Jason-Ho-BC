Overview
Dish Discovery helps meal decision making seamless and stress free. You will be able to browse all sorts of cuisines from around the globe. Be able to set up a personalized profile with preferences, and be handed food recommendations at the touch of your fingertips. Find new recipes and explore new meals to create!
Technologies Used
  • Android Studio
  • Java, Kotlin
  • Amazon Web Services (AWS) Cognito, DynamoDB
  • Figma
  • GitHub
  • Retrofit
  • Spoonacular API
  • Postman
